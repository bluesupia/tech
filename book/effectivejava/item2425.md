## 아이템24. 멤버 클래스는 되도록 static으로 만들어라
* 중첩 클래스
	* 다른 클래스 안에 정의된 클래스
	* 자신을 감싼 바깥 클래스에서만 쓰여야 함
	* 그외 쓰임새가 있으면 톱레벨 클래스로 만들어야 함
* 중첩 클래스의 종류
	* 정적멤버 클래스 - 외부
	* (비정적) 멤버 클래스 - 내부
	* 익명 클래스 - 내부
	* 지역 클래스 - 내부
* 정적멤버 클래스
	* 다른 클래스 안에 선언되고, 바깥 클래스의 private 멤버에도 접근가능
	* 그 외 일반 클래스와 동일
