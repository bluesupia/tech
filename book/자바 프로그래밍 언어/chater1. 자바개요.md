## 1. 시작하기
* 자바프로그램
	* 클래스로 만들고 이를 사용해 클래스 인스턴스인 객체를 생성할 수 있다

	* 객체 : 공장에서 만들어지는 부품

		* 필드 : 클래스나 클래스객체에 속하는 데이터 변수
		* 메소드 : 클래스 행위를 정의
* Hello, World
	* 소스를 선호하는 편집기로 작성하고 파일로 저장
	* 컴파일러로 컴파일해서 바이트 코드를 생성
		* 바이트코드 : JVM이 인식하는 기계어
	* 컴파일한 프로그램 실행
